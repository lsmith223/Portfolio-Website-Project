WELCOME!

This is a mockup website for a personal professional portfolio. This website aims to be more than just a resume: it provides a broader, more detailed information on my professional pursuits in an organized manner while showing off some of my coding skills at the same time. Additionally, the website can be updated as I learn more advanced coding knowledge, and when I graduate I will have a base to use if I want to create my own website or professional portfolio.

On this website, you can find information about me, view my resume or portfolio, and view listed accomplishments. There is also a contact me form that functions as emailing me through a form on website. In its correct state, it is not functionable, but is rather a proof of concept.
